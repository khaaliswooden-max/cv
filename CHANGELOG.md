# Changelog

All notable changes to this curriculum vitae are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project loosely follows [Semantic Versioning](https://semver.org/):
major versions mark significant career milestones, minor versions add new
roles or sections, and patches cover corrections and refinements.
