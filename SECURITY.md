# Security & Privacy Policy

This repository contains a personal curriculum vitae. It is documentation,
not executable software, so "security" here is primarily about **privacy**
and **data integrity**.

## Reporting a Concern

If you discover any of the following, please report it privately rather than
opening a public issue:

- Exposed sensitive personal information (e.g., private contact details,
  identifiers, or credentials) that should not be public.
- Content that appears to have been tampered with or misrepresents the
  repository owner.
- Any use of the material that violates the [LICENSE](./LICENSE).

**How to report:** Contact the repository owner through the professional
channels listed in the [README](./README.md#-contact) (LinkedIn is
preferred for private disclosure). Please do not disclose the concern
publicly until it has been addressed.

## What to Expect

- Acknowledgement of your report as soon as reasonably possible.
- A good-faith effort to review and remediate valid concerns.
- Credit for responsible disclosure, if you would like it.

## Data & Privacy Notes

- Contact information in this repository is provided for legitimate
  professional and hiring purposes only.
- Automated scraping, bulk collection, or use of the personal data here for
  unsolicited marketing is not authorized (see the [LICENSE](./LICENSE)).
- No credentials, secrets, or access tokens should ever be committed to this
  repository. If you find any, please report it immediately.

## Scope

This policy covers the content of this repository only. It does not extend to
third-party platforms (LinkedIn, GitHub, etc.) linked from these documents.
